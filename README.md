# NovaLearn - E-Learning Platform

![NovaLearn](https://img.shields.io/badge/NOVALEARN-E--LEARNING%20PLATFORM-FFD700?style=for-the-badge) ![HTML](https://img.shields.io/badge/HTML-5-E34F26?style=for-the-badge&logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS-3-1572B6?style=for-the-badge&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JAVASCRIPT-ES6%2B-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black) ![Node.js](https://img.shields.io/badge/NODE.JS-RUNTIME-339933?style=for-the-badge&logo=node.js&logoColor=white) ![Express](https://img.shields.io/badge/EXPRESS-5.2.1-000000?style=for-the-badge&logo=express&logoColor=white)

NovaLearn is an interactive E-Learning Platform designed to provide high-quality educational content across various domains like Web Development, Cyber Security, and Data Science. The platform offers structured courses, individual lessons, and built-in quizzes to assess knowledge and track progress.

## Features

- **Interactive Courses**: Explore comprehensive courses with multiple lessons.
- **Video Integration**: Lessons include embedded videos for better understanding.
- **Knowledge Checks**: Built-in quizzes at the end of lessons to test your understanding.
- **Dynamic Content**: Course content and progress are loaded dynamically from the backend API.
- **Responsive Design**: The frontend is built to be accessible and responsive across devices.
- **RESTful API**: Backend built with Express to serve course data and static files.

## Technologies Used

### Frontend
- **HTML5**: For structuring the web pages (`index.html`, `course.html`).
- **CSS3**: For styling and layout (likely embedded or external stylesheets).
- **Vanilla JavaScript**: For dynamic DOM manipulation, handling user interactions, and fetching data from the backend (`script.js`, `course.js`, `dashboard.js`).

### Backend
- **Node.js**: The runtime environment for executing server-side code.
- **Express.js**: A fast, unopinionated, minimalist web framework for Node.js used to build the RESTful API and serve static files.

### Other Tools & Libraries
- **Git & GitHub**: Version control and code hosting.

## Getting Started

### Prerequisites
- Node.js installed on your local machine.

### Installation & Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/sewmi388-commits/NovaLearn.git
   ```
2. Navigate into the project directory (e.g., `E learning Platform`).
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
   *or*
   ```bash
   node server.js
   ```
5. Open your browser and navigate to `http://localhost:3000`.
